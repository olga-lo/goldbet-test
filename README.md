# Задание 1: Исследовательское тестирование


## Основные пользовательские сценарии

|  | Название сценария | Почему важно |
|---|---|---|
| 1 | Регистрация через E-mail (Happy Path) | Если основной сценарий не работает, то форма не выполняет задачу. Проверяет: email, пароль, валюта, бонус, соглашения, редирект в ЛК |
| 2 | Регистрация через Phone | Альтернативный метод, быстрая верификация. Если не работает, то потеря части аудитории в пользу конкурентов |
| 3 | Регистрация через Socials (Google, Facebook, Telegram) | Популярный метод входа для молодой аудитории. Если не работает, то потеря пользователей |
| 4 | Регистрация с применением промокода | Промокоды это канал привлечения пользователей. Если не работает, пользователь не получит бонус |
| 5 | Регистрация с выбором бонуса | Бонус мотивирует пополнить первый депозит. Ошибка в бонусе это прямые финансовые убытки |
| 6 | Регистрация без выбора бонуса | Пользователь имеет право отказаться от бонуса. Если опция не работает то нарушение |
| 7 | Повторная регистрация дубликат Email/телефона | Нельзя создать два аккаунта на один email или телефон. Это защита от мошенничества: один человек не получит два бонуса. Без этой проверки казино потеряет деньги и лицензию |
| 8 | Регистрация без чек-бокса 18+ | Юридическое требование. Гемблинг только для совершеннолетних. Нарушение = штрафы, аннулирование лицензии |

# Задание 4: Баг-репорты формы регистрации 

---

## Баг 1: Отсутствует кнопка Зарегистрироваться после исправления серверной ошибки валидации

**Шаги воспроизведения:**
1. Открыть форму регистрации https://goldvexy.com/registration
2. Заполнить поле Email указав некорректный email
3. Заполнить поле Пароль валидным значением
4. Отметить чекбокс согласия (старше 18 лет)
5. Нажать кнопку Зарегистрироваться
6. Пройти CAPTCHA
7. Получить ошибку о невалидном email
8. Исправить поле Email на валидное значение
9. Нажать кнопку Зарегистрироваться

**Фактический результат:**
Кнопка Зарегистрироваться отсутствует. Вместо неё отображается капча. Пользователь не может завершить регистрацию после исправления ошибки.

**Ожидаемый результат:**
После исправления email кнопка Зарегистрироваться возвращается. Пользователь может повторно отправить форму.

**Severity:** High  
**Priority:** High  

**Окружение:** Браузер Chrome 125, Windows 11

---

## Баг 2: Кнопка Зарегистрироваться остаётся активной при снятии чекбокса согласия

**Шаги воспроизведения:**
1. Открыть форму регистрации https://goldvexy.com/registration
2. Заполнить поле Email валидным email
3. Заполнить поле Пароль валидным значением
4. Снять чекбокс согласия (старше 18 лет)

**Фактический результат:**
Кнопка Зарегистрироваться остаётся активной. После нажатия пропускает на шаг ввода капчи.

**Ожидаемый результат:**
При снятии чекбокса старше 18 лет кнопка Зарегистрироваться должна задизейблиться (стать неактивной). Пользователь не может отправить форму без принятия согласия.

**Severity:** High  
**Priority:** High  

**Окружение:** Браузер Chrome 125, Windows 11

---

## Баг 3: Несоответствие валидации поля промокода при вводе с клавиатуры и при вставке из буфера обмена

**Шаги воспроизведения:**
1. Открыть форму регистрации https://goldvexy.com/registration
2. Нажать кнопку «Do you have a promo code?»
3. Попытаться ввести буквы с клавиатуры в поле промокода
4. Скопировать текст с буквами (например, BONUS100) из буфера обмена
5. Вставить текст в поле промокода

**Фактический результат:**
При вводе с клавиатуры буквы не вставляются, только цифры. При вставке из буфера буквы принимаются.

Периодически иногда можно вводить буквы в поле промокод с клавиатуры, а иногда ввод букв с клавиатуры недоступен. Какой-либо закономерности не выявлено.

**Ожидаемый результат:**
Ввод с клавиатуры и вставка из буфера должны обрабатываться одинаково. Если фильтр по клавиатуре запрещает буквы, вставка тоже должна запрещать. Если вставка разрешена, ввод с клавиатуры тоже должен быть разрешён.

**Severity:** Medium  
**Priority:** Medium

**Окружение:** Браузер Chrome 125, Windows 11

**Комментарий для разработчиков:** Уточнить требования: промокод должен содержать только цифры или буквы тоже допустимы? После уточнения привести оба способа ввода к одному поведению.

# Самые дорогие баги для бизнеса в гемблинге


### 1. Регистрация без подтвердждения возраста 18 лет
Это может привести к юридическим последствиям, штрафам, потери лицензии, медиа-скандалам, компенсациям

### 2. SQL-injection / XSS
Приведет к утечке данных пользователей, краже сессий админов, компрометации всей системы, репутационным потерям

### 3. Дубликаты аккаунтов
Финансовые потери. Один человек получает два приветственных бонуса, при 1000 регистраций в день потери масштабируются


Баг 1 и 2 могут уничтожить бизнес полностью. Баг 3 вызывает постоянные убытки. Все три нужно тестировать в первую очередь.
